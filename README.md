# Математическая библиотека · DS / ML

Сравнительные таблицы учебников по математике для перехода в Data Science и Machine Learning. Шесть разделов — от линейной алгебры до численных методов.

🔗 **[Открыть сайт](https://lepih.github.io/math-library-ds/)**

## Разделы

| # | Дисциплина | Файл |
|---|------------|------|
| 01 | Линейная алгебра | `linear_algebra_books_ds_comparison.html` |
| 02 | Математический анализ | `real_analysis_ds_comparison.html` |
| 03 | Функциональный анализ | `functional_analysis_ml_table.html` |

---

## Что внутри каждой таблицы

- **Рейтинг Goodreads** — реальные оценки читателей
- **Уровень сложности** — начальный / средний / продвинутый
- **Математическая строгость** — визуальная шкала от 1 до 5
- **DS-ценность** — насколько книга полезна именно для DS/ML
- **Фильтрация и поиск** — по тегам, уровню, ключевым словам
- **Сортировка** — по любому столбцу
- **Рекомендация** — когда и кому читать
